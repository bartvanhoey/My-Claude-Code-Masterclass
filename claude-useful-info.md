# Useful Claude Prompts

## prompt at the start of the project

analyze the codebase and provide a          
  summary of this project, outlying any      
  frameworks, architecture or main            
  features  


## Resume previous conversation

```bash
  claude -r (--resume)
```


## Where to find Claude-Session info

on a macmini press SHIFT+ALT + . to see hidden files in the yourusername folder.

Navigate to .claude/projects to see the session info per project


## Switch Model in a Session

```bash
  /model
```

## Show Usage

```bash
  /usage
```

 ## Config Claud

```bash
  /config
```

 ## Claude.md

```bash
  /init
```

## Visualize current context usage as a colored grid


```bash
  /context
```

## Create a summary to the session context

Free up space, create a summary, but possible loss of some context. Runs automatically if space 95% used

```bash
  /compact
```

## Rewind to an earlier conversation

```bash
  /rewind or Esc Esc
```
